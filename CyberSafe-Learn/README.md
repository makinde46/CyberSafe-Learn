# 🛡 CyberSafe Learn

## Interactive Multimedia System for Cybersecurity Awareness

### Project Overview

CyberSafe Learn is an interactive multimedia web application developed as a Multimedia Systems and Applications project. The application educates users about cybersecurity threats and safe online practices using multimedia elements such as text, images, audio, video, animation, and interactive quizzes.

---

## Features

- Interactive Home Page
- About Section
- Cybersecurity Lessons
- Image Gallery
- Video Player
- Audio Narration
- Interactive Quiz
- Automatic Score Calculation
- Certificate of Completion
- Dark Mode
- Responsive Design
- Loading Screen
